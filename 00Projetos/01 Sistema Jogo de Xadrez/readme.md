# Projeto Sistema Jogo de Xadrez
 
O projeto faz parte do Curso: [C# COMPLETO Programação Orientada a Objetos + Projetos](https://www.udemy.com/course/programacao-orientada-a-objetos-csharp/)

Professor [Dr. Nelio Alves](http://educandoweb.com.br)

Indico o curso devido o nível de detalhamento, conteúdo e didática.

##Topicos do projeto:
* Encapsulamento / Modificadores de acesso;
* Construtores;
* ToString;
* Poliformismo;
* Overriding (sobrescrita);
* Associações;
* Enumerações;
* Herança;
* Downcasting;
* Membros estáticos;
* Layers pattern;
* Exceções.

##Projeto:
* Criado tabuleiro;
* Criado peças (Peão, Torre, Bispo, Cavalo, Rainha, Rei);
* Impresso o tabuleiro com as peças;
* Colocado peças no tabuleiro;
* Permitido movimentação das peças;
* Impressão de movimentos possíveis;
* Troca de jogador a cada turno;
* Manipular peças capturas;
* Implantado a lógica do xeque;
* Jogadas especiais (Roque, en passant e promoção).
